Audit the content presentation for: $ARGUMENTS

Look for improvements to:

- page structure and scanability
- prose typography
- code block readability
- tables and lists
- metadata, tags, dates, and reading time
- collection navigation
- mobile behavior
- accessibility

Return a prioritized list of concrete changes with file paths. If the changes are low-risk, implement them and run `npm run build`.
